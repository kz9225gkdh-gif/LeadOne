LeadOne 지원 페이지

LeadOne은 Apple 건강 앱의 Apple Watch ECG 데이터를 가져와 ECG 파형을 확인하고, LVEF screening support 참고 정보를 기기 내에서 제공하는 iOS 앱입니다.

대한민국 출시 준비용 공개 지원 페이지

데이터 처리 방식 ECG 데이터와 분석 결과는 사용자의 기기 내부에서 처리되며, 현재 버전은 서버 업로드와 외부 API 전송을 하지 않습니다.
안내 성격 LeadOne은 ECG 파형 확인과 참고 정보 제공을 위한 앱이며, 의료적 진단이나 치료 판단을 대신하지 않습니다.

앱 소개

LeadOne은 Apple 건강 앱에 저장된 Apple Watch ECG 데이터를 바탕으로 ECG waveform과 metadata를 확인하고, 기기 내부에서 LVEF screening support 참고 정보를 함께 살펴볼 수 있도록 설계된 iOS 앱입니다.

사용자는 저장된 ECG 기록을 다시 확인하고, 앱 안에서 제공되는 모델 정보와 한계 안내를 함께 볼 수 있습니다.

주요 기능

  • Apple 건강 앱의 Apple Watch ECG 가져오기
  • ECG waveform 및 metadata 확인
  • 저장된 ECG 기록 목록 확인
  • 기기 내부 LVEF screening support 참고 분석
  • 모델 정보 및 한계 화면 제공

Apple 건강 앱 ECG 접근 안내

LeadOne은 사용자가 허용한 경우에만 Apple 건강 앱의 ECG 데이터를 읽습니다. 권한을 허용하지 않으면 ECG 가져오기와 관련 화면 이용이 제한될 수 있습니다.

접근 권한은 iOS 설정 또는 Apple 건강 앱에서 언제든지 변경할 수 있습니다. 앱은 Apple 건강 앱의 원본 ECG 데이터를 외부로 전송하지 않습니다.

앱 데이터 처리 방식

  • ECG 데이터와 분석 결과는 사용자의 기기 내부에서 처리됩니다.
  • 서버 업로드를 하지 않습니다.
  • 외부 API로 ECG 데이터를 전송하지 않습니다.
  • 제3자 공유, 광고, 추적 기능을 사용하지 않습니다.

의료적 안내

  • 이 앱은 LVEF를 직접 측정하지 않습니다.
  • 분석 결과는 건강 관리 참고 정보입니다.
  • 실제 평가는 의료기관의 적절한 검사와 전문가 평가를 통해 이루어져야 합니다.
  • 증상이 있거나 결과가 우려되는 경우 의료 전문가와 상담하십시오.
  • 심한 가슴 통증, 갑작스러운 호흡곤란, 실신 등 급한 증상이 있으면 즉시 응급 진료를 받으십시오.
