LeadOne 개인정보 처리방침

이 페이지는 LeadOne이 어떤 건강 데이터를 접근하고, 어떤 목적으로 사용하며, 사용자가 권한과 데이터를 어떻게 관리할 수 있는지 설명합니다.

대한민국 출시 준비용 공개 개인정보 처리방침

핵심 원칙 ECG 데이터와 참고 분석 정보는 현재 버전 기준으로 사용자의 기기 내부에서 처리됩니다.
외부 전송 여부 현재 버전은 서버 업로드, 외부 API 전송, 제3자 공유, 광고, 추적을 하지 않습니다.

접근하는 데이터

  • Apple Watch ECG 기록
  • ECG 파형
  • ECG metadata

LeadOne은 사용자가 허용한 경우에만 Apple 건강 앱의 ECG 데이터를 읽습니다.

사용 목적

  • ECG 표시
  • 저장된 기록 확인
  • 기기 내 참고 분석

데이터 처리 방식

  • 서버 업로드 없음
  • 외부 API 전송 없음
  • 제3자 공유 없음
  • 광고 없음
  • 추적 없음

앱 내부 저장과 삭제

  • 앱 삭제 시 앱 내부 데이터는 함께 삭제될 수 있습니다.
  • Apple 건강 앱의 원본 ECG 데이터는 앱 삭제만으로 삭제되지 않습니다.
  • 원본 ECG 기록 관리는 Apple 건강 앱과 iOS 설정에서 직접 해야 합니다.

권한 철회 방법

사용자는 iOS 설정 또는 Apple 건강 앱에서 LeadOne의 데이터 접근 권한을 언제든지 허용하거나 철회할 수 있습니다.

  • iOS 설정 > 개인정보 보호 및 보안 > 건강 > LeadOne
  • Apple 건강 앱 > 프로필 또는 계정 아이콘 > 앱 > LeadOne

iOS 버전에 따라 실제 경로는 조금 다를 수 있습니다.
