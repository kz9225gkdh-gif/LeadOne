LeadOne 개인정보 처리방침

LeadOne은 사용자가 허용한 Apple 건강 앱의 Apple Watch ECG 데이터를 불러와 ECG 파형과 관련 정보를 확인할 수 있도록 돕는 iOS 앱입니다. 본 개인정보 처리방침은 앱이 어떤 데이터를 접근하고, 어떻게 사용하며, 사용자가 이를 어떻게 관리할 수 있는지 설명합니다.

최종 업데이트: 2026-05-22

핵심 요약 Apple 건강 앱 ECG 데이터는 사용자가 허용한 경우에만 읽으며, 현재 버전은 기기 내부에서만 처리합니다.
외부 전송 여부 현재 버전은 서버 업로드, 외부 API 전송, 광고, 추적, 제3자 공유를 하지 않습니다.

1. 접근하는 데이터

사용자가 허용한 경우 LeadOne은 Apple 건강 앱에서 다음 정보를 읽을 수 있습니다.

  • Apple Watch ECG 기록
  • ECG 파형 측정값
  • ECG 시작 시간과 종료 시간
  • HealthKit에서 제공하는 ECG classification
  • 제공되는 경우 평균 심박수
  • 제공되는 경우 sampling frequency
  • voltage measurement 개수
  • 제공되는 경우 증상 상태 정보

2. 리뷰어용 샘플 ECG 데이터

LeadOne은 실제 Apple 건강 앱 데이터가 없는 환경에서도 앱 기능을 확인할 수 있도록 읽기 전용 샘플 ECG 데이터 모드를 포함할 수 있습니다.

  • 샘플 ECG 데이터는 실제 사용자의 현재 Apple 건강 앱 데이터로 취급하지 않습니다.
  • 샘플 ECG 데이터는 실제 HealthKit 흐름과 분리된 읽기 전용 데모 모드에서만 사용됩니다.
  • 샘플 ECG 데이터는 HealthKit에 다시 쓰지 않습니다.
  • 샘플 ECG 데이터는 서버나 외부 API로 전송하지 않습니다.
  • 앱 안에서 샘플 ECG 데이터임을 명확히 표시합니다.

3. 앱이 생성하거나 저장하는 데이터

LeadOne은 사용자의 기기 내부에 다음 정보를 저장할 수 있습니다.

  • 가져온 ECG 기록
  • 화면 표시를 위한 ECG 파형 데이터
  • ECG metadata
  • 동일 ECG 기록의 중복 저장 방지를 위한 식별 정보
  • 초기 안내 확인 여부 등 앱 사용 상태 정보
  • 현재 빌드에 포함되는 경우 리뷰어용 읽기 전용 샘플 ECG 리소스

앱에 참고 분석 기능이 포함되는 경우에도 ECG 데이터와 관련 정보는 현재 버전 기준으로 기기 내부에서만 처리됩니다.

4. 데이터 사용 목적

  • Apple 건강 앱에서 ECG 기록 가져오기
  • ECG 기록을 기기 내부에 저장하고 다시 표시하기
  • ECG waveform chart와 metadata 표시하기
  • 동일 ECG 기록의 중복 저장 방지
  • 리뷰어용 읽기 전용 샘플 데모 흐름 제공
  • 현재 빌드에 포함되는 경우 기기 내부 참고 분석 기능 제공

5. 기기 내 처리

LeadOne은 ECG 데이터와 앱 내부 생성 정보를 사용자의 기기 내부에서 처리합니다.

  • 서버 업로드 없음
  • 외부 API 전송 없음
  • 제3자 공유 없음
  • 광고 SDK 없음
  • 추적 없음

6. 건강 데이터와 광고

LeadOne은 Apple 건강 앱 데이터와 ECG 관련 정보를 광고, 마케팅, 사용자 프로파일링 또는 추적 목적으로 사용하지 않습니다.

7. 데이터 공유

현재 버전의 LeadOne은 ECG 데이터나 앱 내부 생성 정보를 제3자와 공유하지 않습니다. 서버 기반 계정, 외부 데이터베이스 연동, 외부 연구 업로드 기능도 제공하지 않습니다.

8. 데이터 저장과 삭제

LeadOne은 가져온 ECG 데이터와 앱 동작에 필요한 정보를 iOS 로컬 저장소에 저장할 수 있습니다.

  • 앱을 삭제하면 앱 내부에 저장된 데이터는 함께 삭제될 수 있습니다.
  • 앱을 삭제해도 Apple 건강 앱의 원본 ECG 기록은 삭제되지 않습니다.
  • 원본 ECG 기록 관리는 Apple 건강 앱과 iOS 설정에서 직접 해야 합니다.

9. Apple 건강 앱 권한 관리

사용자는 언제든지 iOS 설정 또는 Apple 건강 앱에서 LeadOne의 건강 데이터 접근 권한을 허용하거나 철회할 수 있습니다.

  • iOS 설정 > 개인정보 보호 및 보안 > 건강 > LeadOne
  • Apple 건강 앱 > 프로필 또는 계정 아이콘 > 앱 > LeadOne

iOS 버전에 따라 실제 경로는 달라질 수 있습니다. 권한을 철회하면 새로운 ECG 기록을 가져오지 못할 수 있습니다.

10. 아동 개인정보

LeadOne은 아동을 대상으로 설계된 앱이 아니며, 아동의 개인정보를 고의로 수집하지 않습니다.

11. 보안

LeadOne은 iOS의 시스템 보호와 로컬 앱 저장소를 사용하여 기기 내부의 앱 데이터를 보호합니다. 다만 어떤 저장 방식도 완전할 수는 없으므로, 사용자는 기기 암호, 생체 인증, 최신 iOS 업데이트 등 기기 보안 설정을 유지하는 것이 좋습니다.

12. 건강 관련 안내

LeadOne은 ECG 데이터를 확인하고 참고 정보를 제공하는 앱입니다. 앱은 의료적 진단이나 치료 판단을 대신하지 않습니다. 증상이 있거나 결과가 걱정되는 경우 의료 전문가와 상담해 주세요.

13. 개인정보 처리방침의 변경

앱의 데이터 처리 방식이 변경되는 경우 본 개인정보 처리방침을 업데이트할 수 있습니다. 변경 시 이 페이지에 최신 버전을 게시하고 최종 업데이트 날짜를 함께 수정합니다.
